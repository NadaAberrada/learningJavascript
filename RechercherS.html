rechercher et supprimer un mot

Enoncé

Ecrire un programme Javascript qui demande à l’utilisateur de saisir
une phrase dans une zone de texte et un mot dans une autre zone de texte.
Le clique sur le bouton rechercher affiche un message si le mot existe dans la phrase ou non.
Le clique sur le bouton supprimer permet d’effacer toutes les occurrence du mot dans la phrase
( si le mot existe bien sûr).
